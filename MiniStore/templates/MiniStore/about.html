About Us

This project was developed by our group as a major requirement for IT 403 WMAD - Elective 5 (WST 3) at Bulacan State University Our goal was to design, build, and present a fully functional Django-based web application. The system features role-based access control with three primary user types: Admin, Seller, and Customer. Its core functionalities include product management (CRUD operations), cart handling using sessions, order summaries, and inventory updates. The application is built with multiple interconnected models (at least 3–4) and uses Django ORM for all database interactions. It also incorporates a responsive user interface, template inheritance, form validation, and complete user authentication—covering signup, login, and logout processes.
